Hide Here

Hide Here is a tiny, focused web app built to help you store short
pieces of text that matter — links, quick to-dos, reminders, copy
snippets, and anything else you want to keep safe in a simple,
distraction-free space.

No noise. No complicated folder systems. Just fast text storage that
gets out of your way so you can get back to what matters.

Features

-   Save short text notes instantly
-   Quickly view stored notes
-   Simple and minimal UI
-   Built using the MVC architecture
-   Lightweight & fast — no bloat

Tech Stack

-   Node.js
-   Express.js
-   EJS (Templating)
-   MVC Architecture

Project Structure

/project ├── controllers/ ├── models/ ├── views/ ├── public/ ├── routes/
├── app.js └── package.json

Installation & Setup

Clone this repository git clone
https://github.com/your-username/hide-here.git

Go into the project folder cd hide-here

Install dependencies npm install

Start the server npm start

App runs at: http://localhost:3000

Usage

1.  Open the app in your browser
2.  Type your text snippet
3.  Save it
4.  Access it anytime later

Goal

Hide Here exists to be a calm place for text — not a productivity suite,
not a dashboard — just a tiny tool that respects your time and focus.
